### 5. "Zero-Training Improvement Loop" — Self-Correction Without GPUs

**Source:** Inversion
**Thread:** Autoresearch loop architecture

Use GEPA-style prompt evolution + MetaClaw skill generation as the primary improvement mechanism. Gate verdicts drive prompt mutations and skill candidates; deterministic gates score them; the loop runs on inference cost alone. Training (SFT/DPO/GRPO) becomes optional amplification, not a prerequisite.

This differentiates from AIDE (tree-search, no gates on corrections), GEPA (prompt evolution, no domain-specific validation), MetaClaw (skill evolution, no governed admission), and Agent-RLVR (guidance injection for sparse rewards — Detrix provides dense per-step signal).

**Warrant:** external: MetaClaw +32% accuracy gradient-free. GEPA outperforms GRPO 6-20% with 35x fewer rollouts. AIDE SOTA on MLE-Bench without fine-tuning. reasoned: Removing training as a hard dependency means customers improve agents on day 1 without GPU infrastructure.

**Action:** YC pitch: "Your agents improve from governed failures on day one. No GPUs, no training pipeline. Gates catch failures, failures become prompt/skill mutations, mutations get scored by the same gates. Training is gear two when you want to bake improvements into weights."

### 7. "Transition-Admission as Legal Admissibility" — The Sharpest Positioning

**Source:** Cross-domain analogy
**Thread:** Moat / architecture

Courts distinguish admissible from inadmissible evidence. The same fact can be true but inadmissible — provenance determines usability. "Fruit of the poisonous tree" = if a gate version changes, all traces scored by the old version are tainted. Precedent = once a decision boundary is validated against holdout data, it compounds for future evaluations.

This maps precisely to VerdictContract's rejection_type: "input_quality" (inadmissible, exclude from training) vs "output_quality" (admissible but rejected, use as DPO negative). Support-query versioning (flush on ANY gate version change) is the governance equivalent of evidence chain of custody.

**Warrant:** reasoned: Federal Rules of Evidence (FRE 401-403) codify truth vs admissibility. Daubert v. Merrell Dow established that expert testimony requires sufficient facts, reliable methods, and reliable application — structurally identical to Detrix gate requirements.

**Action:** Use this framing in the YC app and every pitch: "Hermes makes agent learning visible. Detrix makes agent learning admissible. The difference is the same as the difference between 'this is true' and 'this can be used as evidence in court.'"
